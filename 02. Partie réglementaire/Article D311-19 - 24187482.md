# Article D311-19

L'inventaire des vergers a pour objet de recueillir les informations nécessaires à la connaissance du potentiel de production
arboricole française. Il porte sur les espèces fruitières suivantes : 

a) Pomme ; 

b) Poire ; 

c) Pêche-nectarine ; 

d) Abricot ; 

e) Noix ; 

f) Cerise ; 

g) Prune ; 

h) Raisin de table ; 

i) Kiwi ; 

j) Agrume.
